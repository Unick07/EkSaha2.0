# Nextexa Lab

Full-stack subscription IT services platform.

## Quick start

```bash
npm install
cp client/.env.example client/.env
cp server/.env.example server/.env
npm run dev
```

The frontend runs on `http://localhost:5173` and the API on `http://localhost:5000`.

Demo credentials:

- User: `user@nextexa.dev` / `password`
- Admin: `admin@nextexa.dev` / `password`

Without MongoDB or Stripe credentials, the app runs in demo mode with local data.
